# Asset del template

Metti qui gli asset usati dal template originale.

```text
img/       immagini esportate
fonts/     font locali .woff2, se disponibili
icons/     icone svg/png
```

Se il template usa font esterni ma i file font non sono disponibili, non caricare font remoti senza autorizzazione. Segnala il limite e usa un fallback coerente.
