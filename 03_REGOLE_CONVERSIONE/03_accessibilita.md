# Accessibilità

Requisiti minimi:

- skip link all'inizio della pagina;
- focus visibile;
- menu navigabile da tastiera;
- pulsante menu mobile con `aria-controls` e `aria-expanded`;
- chiusura menu con `Escape` se presente menu mobile;
- form con `label` reali;
- messaggi e placeholder non devono sostituire le label;
- link per navigazione e CTA verso pagine;
- button per azioni JS;
- immagini informative con `alt` descrittivo;
- immagini decorative con `alt=""`;
- contrasto leggibile.

Il codice deve essere comprensibile anche per software di lettura, non solo visivamente simile.
