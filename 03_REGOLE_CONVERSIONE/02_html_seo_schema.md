# HTML, SEO tecnica e schema.org

## HTML

Ogni pagina deve essere un documento completo:

- doctype;
- html lang;
- head completo;
- body organizzato in header, main, footer;
- sezioni semantiche;
- navigazione corretta.

## Heading

- un solo `h1` per pagina;
- `h2` per sezioni principali;
- `h3` per card o sottosezioni.

## SEO tecnica minima

Ogni pagina deve avere:

- title specifico;
- meta description specifica;
- canonical;
- Open Graph base;
- eventuale robots meta solo se necessario.

Le meta description possono essere derivate solo dai testi presenti nel template. Non inventare messaggi commerciali nuovi.

## Schema.org

Inserire JSON-LD coerente, per esempio:

- `Organization`, solo con dati realmente presenti;
- `WebSite`;
- `WebPage` specifico per pagina.

Non inventare indirizzi, telefoni, email, social, partita IVA o dati aziendali.
