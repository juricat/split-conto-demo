# Istruzioni Claude Code

Leggi `PROMPT_AVVIO_CODEX_CLAUDE.md` prima di modificare il progetto.

Non usare Tailwind.
Non creare `src/pages`.
Non inventare testi.

Lavora direttamente sui sorgenti in `05_MINI_SITO/`:

- `index.html`
- `servizi.html`
- `contatti.html`
- `assets/css/styles.css`
- `assets/js/main.js`

Dopo le modifiche esegui:

```bat
npm.cmd run build
```

La cartella finale da caricare online è `05_MINI_SITO/dist/`.
