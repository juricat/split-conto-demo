# Template HTML di input

Metti qui i template HTML esportati da UX Pilot, Figma o strumenti simili.

Esempio:

```text
home.html
servizi.html
contatti.html
```

Questi file sono la fonte unica per testi, CTA, label, card, footer e contenuti visibili.

Il modello deve leggerli e convertirli, non pubblicarli così come sono.
