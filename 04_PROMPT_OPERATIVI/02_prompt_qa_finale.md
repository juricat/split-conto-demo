# Prompt QA finale

Controlla il mini sito dopo la conversione.

Verifica:

- testi identici ai template di input;
- nessun testo inventato;
- un solo `h1` per pagina;
- `h2` e `h3` coerenti;
- title e meta description presenti;
- schema.org senza dati inventati;
- menu e form accessibili;
- CSS custom organizzato;
- JS minimo;
- HTML/CSS/JS minificati in `dist/`;
- nessuna dipendenza Tailwind;
- nessuna CDN non necessaria.

Alla fine riporta solo:

1. cosa è conforme;
2. cosa va controllato manualmente;
3. eventuali errori bloccanti.
